# 서울 지하철 노인 무임승차 — 데이터 저널리즘 웹페이지

기사를 외부 서버(Vercel)에서 공개하는 인터랙티브 신문기사 웹페이지.

## 기술 스택
- **Next.js 16** (App Router, JavaScript) + Noto Sans/Serif KR
- **배포**: GitHub → Vercel 자동배포
- **데이터**: 서울교통공사 역별·시간대별 노인 승하차 CSV(`팀과제` 폴더), 서울시 실시간 OpenAPI

## 진행 순서 (최소버전 먼저 배포 전략)

- [x] **0. 스캐폴드** — Next.js 프로젝트 생성, 기사 본문 + 사진 2장, 인터랙티브 자리(플레이스홀더) 표시, 로컬 빌드/구동 확인
- [x] **1. 배포 파이프라인** — GitHub(NaHoYeol/subway) → Vercel 자동배포 완료
- [x] **2. 인터랙티브 ①** — 유튜브 댓글 워드클라우드 (d3-cloud, 빈도+좋아요 보정, 호버 툴팁)
- [~] **3. 인터랙티브 ②** — 현재 분석 그래프 2장(이미지)로 대체. 추후 인터랙티브화 가능
- [x] **4·5. 인터랙티브 ③·④** — 지하철 경로 노인비중 탐색기 (`SubwayExplorer`)
  - 승·하차역+시간대+평일/주말 입력 → 최대 3개 추천 경로(환승 포함)
  - Leaflet+OSM 지도에 경로를 그라데이션 선으로 표시 (백분위 상위=빨강, 하위=파랑)
  - 경로 평균 노인비중의 "상위 %"와 판정(유독 많음~적음) 표시
  - 데이터: 서울교통공사 2024년 역별·시간대별 노인/전체 승하차 평일·주말 평균 (1~8호선)
  - 데이터 파이프라인: `_crawl/fetch_psgr.py`(공공데이터포털 API 키, 검증용) →
    `build_2024.py`(2024 CSV 집계) → `app/_data/subway.json`

## 기사 속 인터랙티브 마커 (대괄호 [] 위치)
1. (캡션) 현장 사진 → **완료** (station.jpg 히어로, market.jpg 본문)
2. `[유튜브 워드클라우드]` → 인터랙티브 ① **완료**
   - 소스: 유튜브 Shorts `bn9XgKEHxfY` "드디어 폐지되는 지하철 노인 무임승차?" 댓글 1,301개
   - 수집: `_crawl/`(repo 외부) yt-dlp → `build_wordcloud.py`(kiwipiepy 명사추출) → `app/_data/wordcloud.json`
3. `[데이터 모델 제시]` → 인터랙티브 ②
4. `[인터렉티브(서울 각 구간별 노인 하차 인구 실시간 api)]` → 인터랙티브 ③·④
