# 서울 지하철 노인 무임승차 — 데이터 저널리즘 웹페이지

기사를 외부 서버(Vercel)에서 공개하는 인터랙티브 신문기사 웹페이지.

## 기술 스택
- **Next.js 16** (App Router, JavaScript) + Noto Sans/Serif KR
- **배포**: GitHub → Vercel 자동배포
- **데이터**: 서울교통공사 역별·시간대별 노인 승하차 CSV(`팀과제` 폴더), 서울시 실시간 OpenAPI

## 진행 순서 (최소버전 먼저 배포 전략)

- [x] **0. 스캐폴드** — Next.js 프로젝트 생성, 기사 본문 + 사진 2장, 인터랙티브 자리(플레이스홀더) 표시, 로컬 빌드/구동 확인
- [ ] **1. 배포 파이프라인** — git 저장소 → GitHub → Vercel 라이브 URL 확보 *(GitHub/Vercel 로그인 필요)*
- [ ] **2. 인터랙티브 ①** — 유튜브 댓글 워드클라우드
- [ ] **3. 인터랙티브 ②** — 시간대별 노인 지하철 이용 추이 차트 (CSV)
- [ ] **4. 인터랙티브 ③** — 구간/역별 노인 하차 인구 탐색 (CSV)
- [ ] **5. 인터랙티브 ④** — 서울시 실시간 OpenAPI 연동 *(API 키 발급 필요)*

## 기사 속 인터랙티브 마커 (대괄호 [] 위치)
1. (캡션) 현장 사진 → **완료** (station.jpg 히어로, market.jpg 본문)
2. `[유튜브 워드클라우드]` → 인터랙티브 ①
3. `[데이터 모델 제시]` → 인터랙티브 ②
4. `[인터렉티브(서울 각 구간별 노인 하차 인구 실시간 api)]` → 인터랙티브 ③·④

## 보유 자산 경로
- 기사 원문: `../article_extracted.txt` (원본 `_웹페이지 구성안_.docx`)
- 사진 원본: `../docx_extract/word/media/image1.jpg`(시장), `image2.jpg`(역)
- 데이터: `../../팀과제/서울교통공사_역별 일별 시간대별 노인 승하차인원 정보_20241231.csv`
- 기존 분석 차트: `../../팀과제/output/*.png`

## 로컬 실행
```
npm run dev      # http://localhost:3000
npm run build    # 배포 전 빌드 확인
```

## 사용자(너)가 직접 해야 하는 단계
- **1단계**: GitHub 계정 로그인, Vercel 계정(github 연동) 로그인
- **5단계**: 서울 열린데이터광장 API 키 발급
